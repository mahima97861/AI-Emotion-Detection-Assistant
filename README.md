# 🎭 AI Emotion Detection Assistant

## 📌 Overview
This project is a real-time AI-based Emotion Detection System using Computer Vision. It detects human emotions through webcam input and provides feedback via text, voice, and graphs.

## 🚀 Features
- Real-time face detection using OpenCV
- Emotion classification (Happy, Sad, Angry, Surprise, Neutral)
- Voice feedback using Text-to-Speech
- Live emotion trend graph
- Lightweight and fast (no deep learning required)

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- pyttsx3

## 📂 Project Structure

main.py - Main application
requirements.txt - Dependencies
assets/ - Screenshots & outputs
